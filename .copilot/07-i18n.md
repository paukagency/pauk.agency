# 07 — Internationalization (i18n)

## Languages
9 языков: `en`, `ru`, `es`, `de`, `fr`, `hi`, `it`, `pt`, `zh`

## File Structure
```
js/lang/
  en.js  ru.js  es.js  de.js  fr.js  hi.js  it.js  pt.js  zh.js
```

## Format (каждый файл)
```js
window.LANG_XX = {
  key_name: "Translated text",
  ...
};
```

## HTML Usage
```html
<p data-i18n="hero_title">Default text</p>
```
`i18n.js` при загрузке и при смене языка ищет все `[data-i18n]` и подставляет перевод.

## Rules
1. **При добавлении любого текстового элемента** → добавить `data-i18n="ключ"` и перевод во все 9 файлов
2. Ключи — snake_case, описательные: `promo_commission_label`, `step_reg_title`
3. Дефолтный текст в HTML — на русском (для разработки)
4. Плейсхолдеры `<input>` тоже переводятся через `data-i18n`
5. Не хардкодить текст без `data-i18n` (кроме декоративных надписей типа "APP SCREENSHOT")
